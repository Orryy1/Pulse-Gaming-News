# Pulse Enterprise Roadmap

## P0: Safety And Reliability

- Keep production on strict queue dispatch.
- Run system doctor, media verify and platform status before promotion.
- Keep canonical render protected.
- Make queue inspection work against local SQLite and production read-only snapshots.
- Add richer Instagram Story logging.
- Live-proof Facebook Reel ready/complete success.

## P1: Output Quality

- Expand media inventory acquisition for official trailer clips and store assets.
- Use media inventory class to downgrade thin stories automatically.
- Improve thumbnail-safe frame generation with visual contact sheets.
- Keep HyperFrames narrow in the card lane.
- Fix channel-specific audio true-peak warnings before multi-channel promotion.

## P1: Learning Loop

- Connect read-only YouTube Analytics when safe scopes are confirmed.
- Persist snapshots at +1h, +3h, +24h, +72h, +7d and +28d.
- Keep recommendations advisory until sample size is meaningful.
- Connect read-only comments to Comment Copilot without sending replies.

## P2: Formats And Monetisation

- Build Monthly Release Radar from real verified release sources.
- Add Daily Briefing and Weekly Roundup once media inventory is reliable.
- Build newsletter/blog repurposing from fact-checked packages.
- Add YPP and affiliate dashboards.

## Strategic Constraint

The next big unlock is not more render effects. It is better source material and better selection of which stories deserve premium video.
